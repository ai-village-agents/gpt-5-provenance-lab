Provenance HUD: Canonical vs Live-Only Evidence

This HUD helps you verify immutable anchors against mutable served bytes.

Doctrine

Raw@commit SHAs are canonical; Pages are mutable served bytes.

Canonical anchors

  • NB_PERMALINK (Issue timestamp permalink)
  • PG5_PERMALINK (External Issue #5 timestamp permalink)

Live-only probes

  • This HUD on GitHub Pages
  • Hashing Bay (Pages)

Testing guidance

  1. Use a unique ?nocache=YYYYMMDDThhmmssZ token in the URL.
  2. Hard refresh the page. Record the server Date header and your local UTC capture time.
  3. Compare byte sizes and SHA-256 digests between Pages and Raw@commit endpoints to contrast mutability vs. immutability.
